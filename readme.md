# 📊 Bluestock MF Analytics Platform
### Mutual Fund Analytics Capstone · Data Analyst Internship · Cohort 2025

> A full-stack mutual fund analytics platform built on publicly available AMFI India data.
> Covers the complete data pipeline: raw ingestion → ETL → SQL database → EDA → performance analytics → interactive dashboard → advanced risk metrics.

---

## 🚀 Quick Start

```bash
# 1. Clone the repo
git clone https://github.com/riddhima16/bluestock-mf-capstone.git
cd bluestock-mf-capstone

# 2. Install dependencies
pip3 install -r requirements.txt

# 3. Run the complete ETL pipeline (Days 1–4 in one command)
python3 etl_pipeline.py

# 4. Launch the interactive dashboard
streamlit run dashboard.py
```

## ▶️ Running the Project (Day by Day)

### Day 1 — Data Ingestion
```bash
python3 live_nav_fetch.py      # Fetch live NAV from mfapi.in for 6 schemes
python3 data_ingestion.py      # Load + profile all 10 CSVs, validate AMFI codes
```
**Output:** `data/raw/` populated with all CSV files, `reports/data_quality_day1.txt`

### Day 2 — Data Cleaning + SQLite Database
```bash
python3 data_cleaning.py       # Clean all 10 datasets, save to data/processed/
python3 db_loader.py           # Load into SQLite star schema (11 tables)
```
**Output:** `data/db/bluestock_mf.db` with 89,359 rows across 11 tables

### Day 3 — Exploratory Data Analysis
```bash
python3 eda_analysis.py        # Generate 17 charts
python3 create_notebook.py     # Create EDA_Analysis.ipynb
```
**Output:** `reports/charts/01_*.png` through `09_*.png`, `notebooks/EDA_Analysis.ipynb`

### Day 4 — Performance Analytics
```bash
python3 performance_analytics.py        # CAGR, Sharpe, Alpha/Beta, Drawdown, Scorecard
python3 fix_task8.py                    # Benchmark comparison chart
python3 create_performance_notebook.py  # Create Performance_Analytics.ipynb
```
**Output:** `data/processed/fund_scorecard.csv`, `alpha_beta.csv`, `cagr_report.csv`

### Day 5 — Interactive Dashboard
```bash
streamlit run dashboard.py
```
**Opens at:** `http://localhost:8501`
Pages: Industry Overview · Fund Performance · Investor Analytics · SIP & Market Trends

### Day 6 — Advanced Analytics
```bash
python3 advanced_analytics.py           # VaR, Rolling Sharpe, Cohort, HHI
python3 recommender.py                  # Fund recommendations (all 3 risk profiles)
python3 create_advanced_notebook.py     # Create Advanced_Analytics.ipynb
```
**Output:** `data/processed/var_cvar_report.csv`, `cohort_analysis.csv`, `sip_continuity.csv`, `sector_hhi.csv`

### Day 7 — Report + Slides
```bash
python3 create_report.py       # Generate reports/Final_Report.html
python3 create_slides.py       # Generate reports/Presentation.pptx
```

### Or run everything at once:
```bash
python3 etl_pipeline.py        # Runs Days 1–6 sequentially
```

## 🗄️ Database Schema

Star schema with **11 tables** and **89,359 total rows**:

| Table | Type | Rows | Description |
|-------|------|------|-------------|
| `dim_fund` | Dimension | 40 | 40 mutual fund schemes across 10 AMCs |
| `dim_date` | Dimension | 1,826 | Calendar dimension 2022–2026 |
| `fact_nav` | Fact | 46,000 | Daily NAV for all 40 schemes |
| `fact_transactions` | Fact | 32,778 | SIP/Lumpsum/Redemption investor transactions |
| `fact_performance` | Fact | 40 | Pre-computed risk/return metrics |
| `fact_aum` | Fact | 90 | Quarterly AUM by fund house |
| `sip_inflows` | Supporting | 48 | Industry-level monthly SIP data |
| `category_inflows` | Supporting | 144 | Net inflows by fund category |
| `folio_count` | Supporting | 21 | Total MF investor accounts over time |
| `portfolio_holdings` | Supporting | 322 | Equity fund top holdings |
| `benchmark_indices` | Supporting | 8,050 | Nifty 50/100/Midcap150/BSESmallCap daily |

| 9 | Large Cap funds show **>0.85 return correlation** — diversification limited | Correlation matrix |
